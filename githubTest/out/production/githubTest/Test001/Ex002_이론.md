###### 24p
### 연산자

- 단항 연산자 a++
- 이항 연산자 a<b
- 삼항 연산자 a ? b : c
  - 조건 연산자는 자바에서 유일한 3항 연산자이다.
  -  ```
     int x = 1;
     int y = 2;
     // 둘 중 큰 값을 반영
     int a = (x>y) ? x : y ;
     ```
